What is a password generator?

A password generator creates a random string of characters that is far harder to guess or crack than anything a person would come up with on their own — no personal names, no dictionary words, no predictable patterns. This tool lets you choose the exact length and character sets you want, then generates a password using your browser's cryptographically secure random number generator, so every character is chosen independently and unpredictably.

How to use it

1. Drag the Length slider to choose how many characters you want (4–64). 2. Tick or untick Uppercase, Lowercase, Numbers, and Symbols to control which character sets are used — at least one must stay checked. 3. Optionally enable Exclude ambiguous characters to drop easily-confused characters like l, 1, I, and O, 0, which is handy if you'll be typing the password by hand. 4. Click Generate, then Copy to grab it.

How is password strength estimated?

The strength meter estimates the password's entropy in bits, calculated as length × log2(character set size) — a rough measure of how many attempts a brute-force search would need on average. More length and a larger mixed character set both increase this number. This is a simple, informational estimate based on length and character variety; it does not check the result against leaked-password lists or account for how or where you use the password, so treat the label (Weak, Fair, Strong, Very strong) as a helpful guide rather than a security guarantee.

Is my password safe?

Yes. This generator runs entirely in your browser using the Web Crypto API's crypto.getRandomValues() — the same cryptographically secure source used for real security-sensitive code — rather than Math.random(), which is not suitable for generating secrets. Nothing you generate is transmitted, logged, or stored anywhere; once you navigate away, the password only exists wherever you pasted it.

Password tips

Use a different password for every account, and store them in a password manager rather than memorizing or reusing them. Longer passwords are generally stronger than shorter ones with more symbol variety, so when a site allows it, prefer length. Turn on two-factor authentication wherever it's offered, since it protects you even if a password is ever exposed.
