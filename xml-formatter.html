What is an XML formatter?

      
An XML formatter takes minified or messy XML and re-indents it into a clean, readable tree so you can scan nested elements and attributes at a glance. This tool also validates your markup and reports the exact parser error when something is malformed, so you can find and fix the problem quickly.

      
How to use it

      
1. Paste your XML into the box above. 2. Click Beautify to indent it, Minify to strip whitespace between tags, or Validate to check for errors. 3. Click Copy to grab the result.

      
What counts as valid XML?

      
Valid XML must have exactly one root element, every opening tag needs a matching closing tag (or must be self-closed like <br/>), tags must be properly nested without overlapping, attribute values must be quoted, and special characters such as & and < inside text must be escaped as entities (&amp;, &lt;). This tool uses the browser's built-in XML parser, so it flags the same errors a real XML consumer would.

      
Is my data safe?

      
Yes. This formatter runs entirely in your browser using the built-in DOMParser engine. Nothing you paste is ever sent to a server.
