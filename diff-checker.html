What is a text diff checker?

A text diff checker (also called a text comparison tool) compares two blocks of text — an "Original" and a "Changed" version — and shows you exactly what changed between them. Instead of re-reading both versions line by line yourself, the tool aligns matching lines automatically and highlights the ones that were added, removed, or edited, so you can review a change in seconds. This is useful for comparing config files, articles, contracts, code snippets, log output, or any two versions of the same document.

How to use it

1. Paste the original text into the Original box and the new version into the Changed box. 2. Click Compare. 3. Lines that only exist in the original are shown in red as removed, lines that only exist in the changed version are shown in green as added, and unchanged lines are shown in gray for context. 4. With Word-level highlighting enabled, a changed line also shows exactly which words differ within it, not just that the whole line changed. Toggle Ignore whitespace or Ignore case if you want to compare content only, ignoring formatting differences. Click Copy result to grab a plain-text version of the diff, or Sample to see it in action.

How does the comparison work?

This tool splits both texts into lines and computes the longest common subsequence (LCS) of lines between them — the largest set of lines that appears in the same relative order in both versions. Everything outside that common sequence is a change: a line missing from the changed version is a deletion, and a line present only in the changed version is an addition. When a deleted line and an added line sit next to each other, the same LCS technique is applied again at the word level within that pair, so only the specific words that changed are underlined, not the entire line.

Is my data safe?

Yes. The comparison runs entirely in your browser using plain JavaScript — there is no server component at all. Neither the original text nor the changed text is ever uploaded or transmitted anywhere, which makes this safe to use for private documents, internal code, or unpublished drafts.
