What is a SQL formatter?

      
A SQL formatter takes a cramped, one-line, or inconsistently indented query and rewrites it into a clean, readable layout — clauses on their own lines, columns aligned, and keywords cased consistently. Readable SQL is easier to review, diff, and debug. This tool never changes what your query does; it only changes how it looks.

      
How to use it

      
1. Paste your SQL into the box above. 2. Pick your database dialect, indent size, and keyword case. 3. Click Format to beautify it, then Copy to grab the result.

      
Which dialects are supported?

      
Standard SQL, PostgreSQL, MySQL, MariaDB, SQLite, T-SQL (SQL Server), PL/SQL (Oracle), BigQuery, Redshift, Snowflake, Spark SQL, and Db2. Choosing the right dialect helps the formatter handle dialect-specific keywords and syntax correctly.

      
Is my data safe?

      
Yes. This formatter runs entirely in your browser using a self-hosted copy of the open-source sql-formatter library (MIT licensed). Nothing you paste is ever sent to a server.
