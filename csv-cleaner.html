What this CSV cleaner changes

The tool applies only the rules you select. It can remove leading and trailing whitespace, fully blank rows, fully blank columns, and exact duplicate rows. It does not guess whether a value is a date or number, change capitalization, merge similar-looking people, or silently rewrite identifiers.

Why the change report matters: cleanup is safer when you can see how many cells and rows changed. Check the counts and preview before downloading, especially for CRM, accounting, analytics or database imports.

How to clean a CSV file

  1. Open a local CSV/TSV file or paste delimited text.
  2. Keep auto-detect on, or choose the delimiter if the preview is not split correctly.
  3. Select only the cleanup rules you need, then press Clean and preview.
  4. Review the row counts and first 50 rows before copying or downloading.

Quoted commas and multiline fields

The parser recognizes common CSV quoting: commas and line breaks inside a double-quoted field stay inside that field, and doubled quotes become one quote. Output values are quoted again when needed, so a note such as "Call, then email" remains one cell.

Frequently asked questions

Does this CSV cleaner upload my file?

No. The file is read and cleaned locally by JavaScript in your browser. Its contents are never sent to this site or another server.

Can it clean TSV and semicolon-delimited files?

Yes. Auto-detect checks comma, tab, semicolon and pipe delimiters. You can also choose the input and output delimiter manually.

How does duplicate removal work?

It compares complete rows after any selected whitespace trimming. Only exact duplicate rows are removed, and the first occurrence is kept.

Will it change dates, numbers or capitalization?

No. This version does not infer data types or rewrite values. It only applies the cleanup options you select.
